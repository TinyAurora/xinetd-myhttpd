# xinetd-myhttpd

### 项目描述
通过实战编写了一个简单的处理http请求的服务程序，并结合linux下的超级服务器xinetd，使其成为一个简单完整的服务器软件程序，并最终部署在阿里云服务器上，来进一步学习和了解完成一个http服务器的大致流程。
